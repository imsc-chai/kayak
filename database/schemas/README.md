# Database Schemas

This folder contains MongoDB schema definitions for all entities.

## Schemas

- `user.schema.js` - User entity schema
- `flight.schema.js` - Flight entity schema
- `hotel.schema.js` - Hotel entity schema
- `car.schema.js` - Car entity schema
- `billing.schema.js` - Billing entity schema
- `admin.schema.js` - Admin entity schema

Each schema will be implemented in the respective service's models folder and referenced here for documentation.
